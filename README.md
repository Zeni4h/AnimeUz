# AnimeUz
# Anime Information Platform

## Overview

Anime Information Platform is a web application that helps anime fans discover anime information, find where anime can be watched, and stay updated on releases.

The platform does not host or stream anime. Instead, it aggregates information from public APIs and official sources.

## Core Features (Phase 1)

* Anime synopsis
* Ratings
* Genres
* Trailers
* Streaming availability
* Region availability

## Planned Features
