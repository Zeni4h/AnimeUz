# Development Tasks

## Current Sprint (MVP)
## MVP Definition

A user can:

1. Search for anime.
2. Open an anime details page.
3. View synopsis.
4. View ratings.
5. View genres.
6. View trailers.
7. View streaming availability.

The project is considered MVP-complete when all above items work in production.
